# VoleyPro - Marcador Digital 

Este proyecto es una herramienta operativa diseñada para la digitalización de marcadores en partidos de voleibol amateur, sustituyendo las actas de papel tradicionales.

##  Funcionalidades
- **Marcador en tiempo real:** Control de puntos y sets para dos equipos.
- **Persistencia de datos:** Los datos no se pierden al cerrar el navegador (uso de LocalStorage).
- **Guía Técnica:** Secciones informativas sobre posiciones, rotaciones y ejercicios.

##  Instalación y Uso
1. Clona el repositorio: `git clone https://github.com/agonzalezn/Proyecto2_Digitalizacion_AGN.git`
2. Abre el archivo `index.html` en cualquier navegador moderno.
3. ¡Empieza a anotar puntos!
